Hello Git

sudo apt-get install git-core (Ubuntu/Debian)

Install Git on OS X:

curl -O http://kernel.org/pub/software/scm/git/git-1.7.2.2.tar.bz2
tar -jxvf git-1.7.2.2.tar.bz2
cd git-1.7.2.2
./configure --prefix=/usr/local
make
sudo make install
git --version (should return "git version 1.7.2.2")

http://help.github.com/mac-git-installation/ (OS X)

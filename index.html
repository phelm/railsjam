Hello Git

sudo apt-get install git-core (Ubuntu/Debian)

Install Git on OS X:

curl -O http://kernel.org/pub/software/scm/git/git-1.7.2.2.tar.bz2
tar -jxvf git-1.7.2.2.tar.bz2
cd git-1.7.2.2
./configure --prefix=/usr/local
make
sudo make install
git --version (should return "git version 1.7.2.2")

http://help.github.com/mac-git-installation/ (OS X)

Git SCM site - http://git-scm.com/


The Git Cheat Sheet - http://github.com/guides/git-cheat-sheet


Hello Ruby Version Manager (RVM)

(sudo) gem install rvm (OS X/ Linux)

nano ~/.bash_profile
[[ -s $HOME/.rvm/scripts/rvm ]] && source $HOME/.rvm/scripts/rvm
rvm install 1.9.2-preview3
rvm 1.9.2-preview3 --default
